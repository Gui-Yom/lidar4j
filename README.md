### LiDAR4J
A java library to read and manipulate LiDAR scans. 

**/!\\** Currently, there is no support for realtime processing and/or event based scan data.

### Supported hardware
 - [Sick TiM510](https://tinyurl.com/y5ajpv2t) (developped with)
